## React Redux CRUD

React Redux CRUD with upload image and relational data with redux-form, react-table and bootsrap

# Install

- npm install / yarn install 
- npm start / yarn start

# Package Used
- react-router-dom
- react-router
- redux
- react-redux
- redux-devtools-extension
- axios
- bootstrap
- react-table
- redux-form
- redux-thunk
